# Password Manager

This is a simple password manager that allows you to securely store and manage your passwords. 
It uses the Fernet symmetric encryption scheme from the `cryptography` library to encrypt passwords. 

## Getting Started

To run the password manager, make sure you have Python installed. You can install the required 
libraries using pip:

pip install cryptography

Then, run the program using Python:

python password_manager.py


## Features

- Create a new master password or enter an existing one
- View saved passwords
- Add new passwords
- Generate random passwords

## Requirements

- Python 3.x
- cryptography library

## Usage

1. Run the program and follow the on-screen instructions.
2. Enter your master password (or create a new one if it's your first time running the program).
3. Choose an option from the menu to view, add, or generate passwords.
